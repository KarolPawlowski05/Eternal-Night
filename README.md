# 🌙 Eternal Night

**Eternal Night** to dynamiczna gra typu **Roguelite Arena Survival** stworzona w C++. Przetrwaj niekończące się fale przeciwników, rozwijaj swoją postać, zdobywaj potężne ulepszenia i walcz z bossami w świecie pogrążonym w wiecznej nocy.

## ⚔️ Główne Funkcje

* Dynamiczna walka inspirowana gatunkiem Survivor-like
* System poziomów i losowych ulepszeń
* Rozbudowane statystyki (Critical Chance, Dodge, Armor, Lifesteal)
* Bronie pasywne:

  * Magic Wand
  * Fire Aura
  * Orbiting Swords
* Proceduralne generowanie przeszkód
* Efekty poprawiające game feel (Screen Shake, Dash, Vignette)
* Architektura oparta na State Machine

## 👹 Bossowie

Podczas rozgrywki gracz zmierzy się z czterema unikalnymi bossami:

* 🌿 **Root Strangler**
* ⚡ **Storm Wraith**
* 🧛 **Vampire Prince**
* 🪨 **Chaos Golem**

Każde starcie z bossem jest podkreślone specjalnymi efektami wizualnymi, drżeniem kamery oraz zmianą muzyki.

## 🛠️ Debug Mode

Gra zawiera ukryty tryb deweloperski aktywowany klawiszem **F3**.

Dostępne funkcje:

* **G** – God Mode
* **N** – Włącz/Wyłącz spawn przeciwników
* **1-4** – Natychmiastowy spawn wybranego bossa
* Podgląd hitboxów i informacji diagnostycznych
